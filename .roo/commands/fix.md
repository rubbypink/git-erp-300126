Hãy phân tích context (file/folder) đính kèm, dựa theo nội dung yêu cầu và chi tiết lỗi trong tin nhắn của người dùng để tìm các đoạn code gây ra lỗi và đưa ra phương án tối ưu cập nhật sửa chữa code lỗi.
